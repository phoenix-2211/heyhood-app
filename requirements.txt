======================================================================
                  HEY HOOD - SYSTEM REQUIREMENTS
======================================================================

To run the Citizen App (hey_hood), Officials App (hood_officials), 
and perform database seeding / local agent operations, the following 
software environment must be installed:

1. Flutter & Dart SDK
---------------------
* Flutter SDK: Version 3.19.x or higher
* Dart SDK: Version 3.3.x or higher (included automatically in Flutter)
* Web Browser: Google Chrome (for running client apps via 'flutter run -d chrome')

2. Node.js Environment (Database Seeding)
-----------------------------------------
* Node.js: Version 18.x or 20.x LTS
* Package Manager: npm (installed with Node.js)
* Required node modules for seeding (installed via npm install):
  - firebase-admin
  - @google/genai

3. Firebase Command Line Interface (Firestore Configuration)
-----------------------------------------------------------
* Firebase CLI (firebase-tools): To deploy or configure security rules
  - Install via npm: npm install -g firebase-tools

4. Python Environment (Optional - Local Agent Development)
----------------------------------------------------------
* Python: Version 3.10.x or 3.11.x
* Required libraries (if running local agent microservices):
  - fastapi, uvicorn, requests, google-generativeai

5. Android SDK & Build Tools (Optional - Building from source)
-------------------------------------------------------------
* Java Development Kit (JDK): openjdk 17 or 21
* Android Studio / Command Line Tools: To build release APKs locally
